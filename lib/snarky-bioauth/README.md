# snarky-bioauth

BioAuth tools for SnarkyJS.

This library provides utilities for interacting with a **zkBioAuth Oracle** and
working with bioAuth ZK proofs within Mina Smart Contracts.

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
